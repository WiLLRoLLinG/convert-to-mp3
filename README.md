
# مبدل صوتی پیشرفته با FFmpeg و پایتون (Advanced Audio Converter)

این اسکریپت پایتون به شما کمک می‌کند تا فایل‌های صوتی خود را از فرمت‌های مختلف به فرمت MP3 با کیفیت بالا تبدیل کنید. این ابزار از کتابخانه قدرتمند **FFmpeg** برای پردازش صدا استفاده می‌کند و یک رابط کاربری ساده در ترمینال به همراه نوار پیشرفت (Progress Bar) برای نمایش روند تبدیل ارائه می‌دهد.

## ویژگی‌ها:

*   پشتیبانی از طیف گسترده‌ای از فرمت‌های صوتی ورودی (مانند MP3, WAV, AAC, FLAC, OGG و غیره).
*   تبدیل به فرمت MP3 با قابلیت تنظیم بیت‌ریت (Bitrate).
*   امکان تعیین پوشه‌های ورودی و خروجی دلخواه.
*   ایجاد خودکار پوشه خروجی در صورت عدم وجود.
*   نمایش نوار پیشرفت کلی برای مشاهده روند تبدیل فایل‌ها.
*   گزارش خلاصه تعداد فایل‌های موفق و ناموفق در انتهای فرآیند.
*   نادیده گرفتن فایل‌هایی که قبلاً تبدیل شده‌اند (در صورت وجود فایل خروجی).
*   عدم انتقال متادیتای فایل‌های صوتی برای اطمینان از سازگاری حداکثری.

## پیش‌نیازها (Prerequisites)

برای استفاده از این اسکریپت، دو پیش‌نیاز اصلی وجود دارد:

1.  **نصب FFmpeg:**
    *   **FFmpeg چیست؟** FFmpeg یک ابزار خط فرمان بسیار قدرتمند و محبوب برای پردازش فایل‌های چندرسانه‌ای (صوتی و تصویری) است. این اسکریپت از FFmpeg برای انجام عملیات تبدیل صدا استفاده می‌کند.
    *   **چگونه نصب کنیم؟**
        *   **وب‌سایت رسمی:** بهترین راه برای دریافت آخرین نسخه FFmpeg مراجعه به وب‌سایت رسمی آن است: [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
        *   **سیستم‌عامل‌ها:**
            *   **Windows:** فایل اجرایی FFmpeg را دانلود کرده و پوشه `bin` آن را به متغیر محیطی `PATH` سیستم خود اضافه کنید. (آموزش‌های آنلاین زیادی برای این کار وجود دارد، کافیست عبارت "add ffmpeg to path windows" را جستجو کنید).
            *   **macOS:** از طریق Homebrew: `brew install ffmpeg`
            *   **Linux (Debian/Ubuntu):** `sudo apt update && sudo apt install ffmpeg`
            *   **Linux (Fedora/CentOS):** `sudo dnf install ffmpeg` یا `sudo yum install ffmpeg`
   
    *   **تست نصب:** پس از نصب، ترمینال یا Command Prompt را باز کنید و دستور زیر را وارد کنید:

        ```bash
        ffmpeg -version
        ```
        
        اگر FFmpeg به درستی نصب شده باشد، اطلاعات نسخه آن نمایش داده می‌شود. در غیر این صورت، باید مشکل نصب یا اضافه کردن به PATH را برطرف کنید.

2.  **نصب پایتون:**
    *   **چگونه نصب کنیم؟**
        *   **وب‌سایت رسمی:** پایتون را از وب‌سایت رسمی آن دانلود کنید: [https://www.python.org/downloads/](https://www.python.org/downloads/)
        *   **هنگام نصب (Windows):** حتماً گزینه "Add Python to PATH" را هنگام نصب تیک بزنید.
        *   **macOS:** پایتون معمولاً به صورت پیش‌فرض نصب است، اما برای اطمینان می‌توانید آخرین نسخه را از سایت رسمی یا طریق Homebrew (`brew install python`) نصب کنید.
        *   **Linux:** معمولاً پایتون از قبل نصب شده است. برای اطمینان از وجود آن، دستور `python3 --version` را اجرا کنید. اگر نیاز به نصب دارید، از مدیر بسته توزیع خود استفاده کنید (مثلاً `sudo apt install python3` برای اوبونتو).
        *   
    *   **تست نصب:** در ترمینال دستور زیر را اجرا کنید:

        ```bash
        python --version
        ```
        
        یا

        ```bash
        python3 --version
        ```
        
        نسخه پایتون شما باید نمایش داده شود.

3.  **نصب کتابخانه `tqdm`:**
    *   این کتابخانه برای نمایش نوار پیشرفت در ترمینال استفاده می‌شود.
    *   **چگونه نصب کنیم؟** ترمینال را باز کنید و دستور زیر را اجرا کنید:

        ```bash
        pip install tqdm
        ```
        یا اگر از `pip3` استفاده می‌کنید:

        ```bash
        pip3 install tqdm
        ```

## نحوه دریافت و اجرای اسکریپت

1.  **دریافت کد اسکریپت:**
    *   کد پایتون ارائه شده را در یک فایل با نام `audio_converter.py` ذخیره کنید.
    *   (اگر از گیت‌هاب دانلود می‌کنید، این فایل به همراه فایل README.md در پوشه شما قرار خواهد گرفت.)

2.  **آماده‌سازی فایل‌های ورودی:**
    *   یک پوشه در سیستم خود ایجاد کنید (مثلاً `my_audio_files`) و تمام فایل‌های صوتی که می‌خواهید تبدیل کنید را در این پوشه قرار دهید.

3.  **اجرای اسکریپت:**
    *   ترمینال (Command Prompt, PowerShell, Terminal) را باز کنید.
    *   با استفاده از دستور `cd` به مسیری بروید که فایل `audio_converter.py` را در آن ذخیره کرده‌اید. مثال:

        ```bash
        cd path/to/your/script/folder
        ```
    *   دستور زیر را برای اجرای اسکریپت وارد کنید:

        ```bash
        python audio_converter.py
        ```
        یا اگر دستور `python` به نسخه 2 پایتون اشاره دارد:

        ```bash
        python3 audio_converter.py
        ```

4.  **پاسخ به سوالات اسکریپت:**
    *   اسکریپت از شما دو مسیر را می‌پرسد:
        *   **مسیر پوشه ورودی (Input Directory):** پوشه‌ای که فایل‌های صوتی اصلی را در آن قرار دادید. اگر Enter را بزنید، از یک مسیر پیش‌فرض (معمولاً پوشه‌ای به نام `input_audio` در همان محل اسکریپت) استفاده می‌کند. **مقدار پیش‌فرض را وارد کنید یا مسیر دلخواه خود را بنویسید** (مثلاً: `C:\Users\YourUser\my_audio_files` یا `/home/youruser/my_audio_files`).
        *   **مسیر پوشه خروجی (Output Directory):** پوشه‌ای که می‌خواهید فایل‌های MP3 تبدیل شده در آن ذخیره شوند. اگر Enter را بزنید، از یک مسیر پیش‌فرض (معمولاً پوشه‌ای به نام `output_audio` در همان محل اسکریپت) استفاده می‌کند. **مقدار پیش‌فرض را وارد کنید یا مسیر دلخواه خود را بنویسید.**
    *   **نکته:** نیازی نیست پوشه‌های ورودی و خروجی حتماً از قبل وجود داشته باشند؛ اسکریپت پوشه خروجی را در صورت عدم وجود، به صورت خودکار ایجاد می‌کند.

5.  **مشاهده روند تبدیل:**
    *   پس از وارد کردن مسیرها، اسکریپت شروع به پردازش فایل‌ها می‌کند.
    *   یک نوار پیشرفت در ترمینال نمایش داده می‌شود که درصد پیشرفت کلی و تعداد فایل‌های پردازش شده را نشان می‌دهد.
    *   تمام پیام‌های مربوط به موفقیت یا شکست تبدیل هر فایل، به صورت خودکار در ترمینال نمایش داده می‌شوند.

6.  **مشاهده نتایج:**
    *   پس از اتمام پردازش تمام فایل‌ها، خلاصه‌ای از نتایج شامل تعداد کل فایل‌ها، تعداد موفق و تعداد ناموفق نمایش داده می‌شود.
    *   فایل‌های MP3 تبدیل شده را می‌توانید در پوشه خروجی که مشخص کرده‌اید، پیدا کنید.

## تنظیمات قابل تغییر در اسکریپت

شما می‌توانید با ویرایش بخش `--- تنظیمات ---` در ابتدای فایل `audio_converter.py`، رفتار اسکریپت را تغییر دهید:

*   **`SUPPORTED_INPUT_FORMATS`**: لیستی از پسوند فایل‌هایی که اسکریپت به عنوان ورودی تشخیص می‌دهد. می‌توانید پسوندهای بیشتری اضافه یا حذف کنید.
*   **`OUTPUT_FORMAT`**: فرمت نهایی فایل‌های خروجی (پیش‌فرض: `.mp3`).
*   **`AUDIO_BITRATE`**: بیت‌ریت فایل‌های MP3 خروجی. مقادیر رایج عبارتند از `'128k'`, `'192k'`, `'256k'`, `'320k'`. مقدار بالاتر به معنی کیفیت بهتر و حجم بیشتر است.

## عیب‌یابی (Troubleshooting)

*   **خطای `ffmpeg not found`:**
    *   مطمئن شوید FFmpeg را به درستی نصب کرده‌اید.
    *   بررسی کنید که پوشه `bin` FFmpeg به `PATH` سیستم شما اضافه شده باشد.
    *   کامپیوتر خود را ری‌استارت کنید تا تغییرات `PATH` اعمال شود.
    *   دوباره دستور `ffmpeg -version` را در ترمینال اجرا کنید.
*   **خطای `python: command not found` یا `python3: command not found`:**
    *   مطمئن شوید پایتون را به درستی نصب کرده‌اید و گزینه "Add Python to PATH" را هنگام نصب انتخاب کرده‌اید.
*   **خطای `ModuleNotFoundError: No module named 'tqdm'`:**
    *   دستور `pip install tqdm` (یا `pip3 install tqdm`) را دوباره اجرا کنید.
*   **فایل‌های خروجی پخش نمی‌شوند یا مشکل دارند:**
    *   ممکن است فایل صوتی ورودی خراب باشد.
    *   ممکن است FFmpeg به درستی نصب نشده باشد.
    *   یک فایل صوتی ساده (مانند یک MP3 کوتاه) را تست کنید تا مطمئن شوید مشکل از اسکریپت نیست.
    *   می‌توانید پارامتر `-loglevel error` را در تابع `convert_audio_file` به `warning` یا `info` تغییر دهید تا هشدارهای بیشتری از FFmpeg دریافت کنید و مشکل را بهتر تشخیص دهید.
