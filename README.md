# مبدل صوتی پیشرفته با FFmpeg و پایتون (Advanced Audio Converter)

این اسکریپت پایتون به شما کمک می‌کند تا فایل‌های صوتی خود را از فرمت‌های مختلف به فرمت MP3 با کیفیت بالا تبدیل کنید. این ابزار از کتابخانه قدرتمند **FFmpeg** برای پردازش صدا استفاده می‌کند و یک رابط کاربری ساده در ترمینال به همراه نوار پیشرفت (Progress Bar) برای نمایش روند تبدیل ارائه می‌دهد.

## ویژگی‌ها:

*   پشتیبانی از طیف گسترده‌ای از فرمت‌های صوتی ورودی (مانند MP3, WAV, AAC, FLAC, OGG و غیره).
*   تبدیل به فرمت MP3 با قابلیت تنظیم بیت‌ریت (Bitrate).
*   امکان تعیین پوشه‌های ورودی و خروجی دلخواه.
*   ایجاد خودکار پوشه خروجی در صورت عدم وجود.
*   نمایش نوار پیشرفت کلی برای مشاهده روند تبدیل فایل‌ها.
*   گزارش خلاصه تعداد فایل‌های موفق و ناموفق در انتهای فرآیند.
*   نادیده گرفتن فایل‌هایی که قبلاً تبدیل شده‌اند (در صورت وجود فایل خروجی).
*   عدم انتقال متادیتای فایل‌های صوتی برای اطمینان از سازگاری حداکثری.

## پیش‌نیازها (Prerequisites)

برای استفاده از این اسکریپت، دو پیش‌نیاز اصلی وجود دارد:

1.  **نصب FFmpeg:**
    *   **FFmpeg چیست؟** FFmpeg یک ابزار خط فرمان بسیار قدرتمند و محبوب برای پردازش فایل‌های چندرسانه‌ای (صوتی و تصویری) است. این اسکریپت از FFmpeg برای انجام عملیات تبدیل صدا استفاده می‌کند.
    *   **چگونه نصب کنیم؟**
        *   **وب‌سایت رسمی:** بهترین راه برای دریافت آخرین نسخه FFmpeg مراجعه به وب‌سایت رسمی آن است: [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
        *   **سیستم‌عامل‌ها:**
            *   **Windows:** فایل اجرایی FFmpeg را دانلود کرده و پوشه `bin` آن را به متغیر محیطی `PATH` سیستم خود اضافه کنید. (آموزش‌های آنلاین زیادی برای این کار وجود دارد، کافیست عبارت "add ffmpeg to path windows" را جستجو کنید).
            *   **macOS:** از طریق Homebrew: `brew install ffmpeg`
            *   **Linux (Debian/Ubuntu):** `sudo apt update && sudo apt install ffmpeg`
            *   **Linux (Fedora/CentOS):** `sudo dnf install ffmpeg` یا `sudo yum install ffmpeg`
    *   **تست نصب:** پس از نصب، ترمینال یا Command Prompt را باز کنید و دستور زیر را وارد کنید:

ffmpeg -version
